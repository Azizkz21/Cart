# React Shopping Cart App

This is a simple React + TypeScript  project built with Vite that demonstrates how to create a small e-commerce store with a shopping cart.  
It uses React Context API for global state management and React Bootstrap for layout and styling.


##  Features

-  Multi-page navigation (Home, Store, About) using - React Router
-  Product listing with data loaded from a JSON file
-  Add, remove, and update items in the shopping cart
-  Automatic total price calculation
-  Responsive navigation bar with a dynamic cart icon
-  Fast and modern setup using - Vite


- React + TypeScript
- Vite
- React Router
- React Bootstrap
- Context API
- ESLint + Prettier (optional for code formatting)
